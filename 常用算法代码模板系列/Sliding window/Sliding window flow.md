# 滑动窗使用思路(寻找最长)

## 1. 左右双指针(L，R)在起始点，R向右逐位滑动
## 2. R对应元素满足条件时，R向右扩大窗口，并更新最优结果
## 3. R对应元素不满足条件时，L向右缩小
## 4. R到达结尾
